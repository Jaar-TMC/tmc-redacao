# TMC Brand Guide

## Overview
TMC is a Brazilian news portal with a clean, modern design focused on readability and visual hierarchy. The brand uses a bold color scheme combining orange and green to create a distinctive and energetic identity.

---

## Color Palette

### Primary Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| TMC Orange | `#E87722` | rgb(232, 119, 34) | Primary accent, CTAs, section headers, highlights |
| TMC Dark Green | `#1A4D2E` | rgb(26, 77, 46) | Navigation bar, secondary accents |
| TMC Light Green | `#2D5A3D` | rgb(45, 90, 61) | Hover states, secondary elements |

### Neutral Colors

| Color Name | Hex Code | RGB | Usage |
|------------|----------|-----|-------|
| White | `#FFFFFF` | rgb(255, 255, 255) | Backgrounds, text on dark surfaces |
| Off-White | `#F5F5F5` | rgb(245, 245, 245) | Section backgrounds, cards |
| Dark Gray | `#333333` | rgb(51, 51, 51) | Headlines, primary text |
| Medium Gray | `#666666` | rgb(102, 102, 102) | Body text, descriptions |
| Light Gray | `#E0E0E0` | rgb(224, 224, 224) | Borders, dividers |

### Accent Colors

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Live Red | `#E53935` | Live indicator dot |
| Alert Orange | `#FF6B35` | Breaking news, alerts |

## Typography

### Logo Font
- **Font Family:** Custom condensed sans-serif (similar to Impact or Oswald)
- **Style:** Bold, uppercase
- **Color:** Orange (`#E87722`)

### Headlines (H1-H2)
- **Font Family:** Arial, Helvetica, sans-serif (or similar system font)
- **Weight:** Bold (700)
- **Size H1:** 24-28px
- **Size H2:** 18-22px
- **Color:** Dark Gray (`#333333`)
- **Line Height:** 1.2-1.3

### Section Headers
- **Font Family:** Sans-serif
- **Weight:** Bold (700)
- **Style:** Uppercase
- **Size:** 14-16px
- **Color:** White on Orange background
- **Letter Spacing:** 0.5px

### Body Text
- **Font Family:** Arial, Helvetica, sans-serif
- **Weight:** Regular (400)
- **Size:** 14-16px
- **Color:** Medium Gray (`#666666`)
- **Line Height:** 1.5-1.6

### Navigation
- **Font Family:** Sans-serif
- **Weight:** Medium (500)
- **Size:** 13-14px
- **Color:** White
- **Text Transform:** None (sentence case)
